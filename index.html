How to open your web browser's JavaScript console

  • Chrome:
    • Windows: Ctrl + Shift + J
    • Mac: Cmd + Opt + J
  • Firefox:
    • Windows: Ctrl + Shift + K
    • Mac: Cmd + Opt + K
  • Safari / Mac: Cmd + Opt + C
    • Note: first you may need to enable web inspector (see instructions here)
