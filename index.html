VARIABLES

What is a variable?

A variable is a piece of information that can change over time


		var myName = 'Ms Awesome';

Copy and paste that into your console, hit enter to run the code. Then type "myName" and hit enter.

NOTE: we show that on the projector in the console too

Things to know about variables:

  • When you make a new variable, define it with "var" written before it.
  • They can contain different kinds of values, or none at all (undefined).
  • A new variable needs to have a unique name.
  • Variable can be named almost anything.
  • Variable names are CASE SENSITIVE and cannot have spaces.

Creating New Variables

Variables can be created with no value (undefined), or with a value:


var favoriteAnimal;
var favoriteColor = 'blue';

Changing the Value of a Variable


var favoriteFood = 'burgers';
favoriteFood = 'pizza';

Try it yourself in your console!

Variables are saved to memory


		favoriteAnimal = "penguin";

Try it yourself in your console!

Variable Types: Strings, numbers, etc


var verb = 'running';
var number = 3;

Plus more, which we'll talk about later!

A couple notes about strings

  • Strings are always surrounded by quote marks such as ' ' or " "
  • A number within a string is not a number, it is text, thus part of the string

Fancy name for it: concatenation!


		var sentence = 'My name is ' + myName + ' and I rock!';

Working with JavaScript Files

- make a folder called madlibs - create an HTML file, index.html - create a JS file, madlibs.js - point out it's best to keep them in the same folder

How to Link Your JavaScript file to your HTML file

Copy this into your index.html file:


<!DOCTYPE html>
<html>
  <head>
    <title>Mad Libs!</title>
    <script src="madlibs.js"></script>
  </head>
  <body>
  </body>
</html>
