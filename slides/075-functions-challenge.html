Challenge: Function practice

  1. Define a function called sayHello that takes no input parameters and returns as output a string that says hello to the world.
  2. Define a function called prettify that takes one parameter, adds "* ~ * ~ * ~" in front of it and "~ * ~ * ~ *" to the end of it, and returns the combination as the output.
  3. Call the sayHello function and save its output in a new variable called greeting.
  4. Call the prettify function, passing it your greeting variable as an argument, and save its output back into the same variable greeting.
  5. Display this final result in the console. Beautiful!
