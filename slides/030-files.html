Working with JavaScript Files

- make a folder called madlibs - create an HTML file, index.html - create a JS file, madlibs.js - point out it's best to keep them in the same folder

How to Link Your JavaScript file to your HTML file

Copy this into your index.html file:


<!DOCTYPE html>
<html>
  <head>
    <title>Mad Libs!</title>
    <script src="madlibs.js"></script>
  </head>
  <body>
  </body>
</html>
